# Nuclear Chemistry — Terminology

## Terms

| Term | Controlled definition | Source | Boundary note |
| --- | --- | --- | --- |
| nuclide | Species of atom characterized by mass number, atomic number, and nuclear energy state, subject to the lifetime boundary in the source definition | NUC-002 | An element name alone does not identify a nuclide |
| isotope | One of a set of nuclides having the same atomic number but different mass numbers | NUC-003 | Isotope is a relation within one element; nuclide is the individual nuclear species |
| nuclear decay | Spontaneous nuclear transformation | NUC-004 | The definition does not specify decay mode or daughter products |
| radioactivity | Property of certain nuclides of undergoing radioactive decay | NUC-001 | Do not use as a synonym for radiation or dose |
| activity | Number of nuclear decays in a small time interval divided by that interval | NUC-005 | Detector count rate is not automatically activity |
| decay constant | Probability-per-time quantity that relates activity to the number of decaying nuclei | NUC-006 | It is state- and process-specific |
| half-life | Time required for activity to decrease to half its value for one radioactive decay process | NUC-007 | It is not the lifetime of each individual nucleus |
| evaluated nuclear data | Recommended nuclear structure or decay information produced through an evaluation system | NUC-009 | Preserve database and retrieval version for reproducibility |
| count | Detector response event retained under a defined acquisition and analysis rule | NUC-014 | Counts depend on instrument, threshold, geometry, and processing |
| live time | Acquisition time during which the measuring system is able to accept events | NUC-014 | State whether corrections for dead time are applied |
| radioisotope | Unstable form of an element that emits radiation while transforming toward a more stable form | NUC-015 | The chemistry required for application is separate from nuclear production |
| disused sealed radioactive source | Sealed source no longer used, and not intended to be used, for its authorized practice | NUC-017; NUC-018 | Disused, spent, stored, and disposed are distinct lifecycle states |
