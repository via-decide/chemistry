# Visual Plan

## Visual system rules

- One dominant visual per spread. Secondary icons, legends, and micro-insets may annotate but may not become a second explanatory diagram.
- Every main visual has a unique `FIG-###` identifier and a unique scientific subject. Reuse is limited to common visual grammar, never the same diagram.
- No illustration is treated as verified merely because it is visually plausible. Molecular, mechanistic, quantitative, apparatus, and process visuals require the accuracy gate listed below.
- Decorative gradients, unlabeled orbitals, impossible glassware connections, unbalanced reactions, and nonphysical scale mixtures are prohibited.
- The repository contains no reusable image assets; all 184 dominant visuals are custom commissions or code-native technical drawings.
